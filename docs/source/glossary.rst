.. _glossary:

=====================================================
 Glossary of Terms Used in OreSat Linux Documentation
=====================================================

.. glossary::
    :sorted:

    OreSat
        PSAS's open source CubeSat
    PSAS
        Portland State Aerosapce Society. A student aerospace group at Portland State University.
    DBus
        Inter-process communication system provided by systemd.
    DKMS
        Dynamic Kernel Module Support. Automatically reboot out of tree Linux kernel modules.
