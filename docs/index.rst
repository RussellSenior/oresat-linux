What is OreSat?
---------------

OreSat is a open source CubeSat from Oregon by PSAS (Portland State Aerospace
Society).

.. note::

    If you are looking for more info on OreSat, go checkout  
    `OreSat Website`_ and `OreSat GitHub`_
